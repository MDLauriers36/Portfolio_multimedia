Here are some of my projects

  • Google In-Store

    Html, CSS, JavaScript, Vue, Vuex

    Google In-Store is a platform allowing Google retail employees to report any issues with their store products. Once an issue is reported, it is posted to an API which populates a separate platform's frontend - which is used by Google's vendors to fix the issue as soon as possible. I was the sole front-end developer of this application.

  • Frontend Simplified

    Html, CSS, JavaScript, Vue, Vuex, Nuxt, Firebase

    Frontend Simplified is a side project built to not only teach people how to code, but teach them how to get job ready in the quickest way possible. Users of this platform can join my 2,300+ member community including engineers from Google, Canva, Atlassian, Apple, Microsoft and Salesforce. The entire platform and courses was built by myself and I'm planning to release the MVP in the next few weeks.
